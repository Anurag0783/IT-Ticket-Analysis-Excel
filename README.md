🎫 IT Ticket Analysis Dashboard (Microsoft Excel)
📌 Project Overview
The IT Ticket Analysis Dashboard is an interactive Microsoft Excel project designed to analyze IT support ticket data and provide actionable business insights. The dashboard helps monitor ticket trends, support team performance, customer satisfaction, and issue resolution efficiency using Excel's advanced analytical features.
This project demonstrates practical data analysis skills by transforming raw ticket data into meaningful visualizations and business recommendations.
________________________________________
🎯 Business Problem
IT support teams receive thousands of service requests every month. Without proper analysis, it becomes difficult to identify performance bottlenecks, monitor customer satisfaction, and improve response efficiency.
This project aims to answer key business questions such as:
•	Which ticket categories receive the highest number of requests?
•	Which support agents perform the best?
•	Which issues take the longest time to resolve?
•	How has ticket volume changed over time?
•	How satisfied are customers with the support provided?
